# 📈 Analytics Case Studies  
Optimizing decisions through rigorous modeling, measurement, and visualization.

---

## 🎓 OSU Entry-Level Placement Assessment (ELPA)

**Client Context**  
OSU’s legacy placement methods leaned heavily on standardized tests, creating inequities and excess remediation.

**Approach**  
Introduced the Predicted Grade Index (PGI) using regression models with high school GPA, class rank, and test scores. Embedded in enrollment workflows and recalibrated routinely.

**Impact**  
Transformed placement accuracy, assessing over 4,400 students annually with remediation rates below 5%. Adapted to students without test scores to extend equity.

## 📊 Key Driver Analysis for Program ROI Measurement

**Client Context**  
Post-financial crisis, 75 programs needed standardized ROI metrics to justify funding.

**Approach**  
Built panel data regression with advisor, client, and market controls. Defined “net flows” and branded as Key Driver Analysis (KDA).

**Impact**  
Enabled program comparisons and capital allocation based on empirical value. Became the core evaluation framework across lines of business.

## 💼 Measuring Wholesaler Impact on Financial Outcomes

**Client Context**  
Firm leadership lacked visibility into how wholesalers affected net flows and product sales.

**Approach**  
Created dynamic panel regression with DAGs and zero-inflated Poisson models. Reproduced in Python for cloud deployment.

**Impact**  
Attributed 22% of net flows to wholesaler engagement. Validated team ROI and guided resource allocation.

## 📊 Transforming Net Flow Reporting with Dashboards

**Client Context**  
Static reporting delayed insight generation and hindered exploratory analysis.

**Approach**  
Built Tableau, Power BI (with Sankey), and Plotly dashboards, integrating data dimensions for geography, advisor, and time.

**Impact**  
Reduced ad hoc analysis demand. Empowered real-time exploration and strategic decision-making across the enterprise.
